# Googleスプレッドシート データベース設定ガイド

このアプリは、Firebaseの代わりにGoogleスプレッドシートをデータベースとして使用します。以下の手順で設定を行ってください。

## 1. スプレッドシートの作成
1. [Google スプレッドシート](https://sheets.new)を新規作成します。
2. 1行目に以下のヘッダー（項目名）を入力してください：
   `name`, `category`, `quantity`, `threshold`, `max`, `unit`, `notes`
3. 適当な名前（例：「ストック管理DB」）を付けて保存します。

## 2. Google Apps Script (GAS) の設定
1. スプレッドシートのメニューから **「拡張機能」** > **「Apps Script」** を開きます。
2. エディタに `backend.gs` の内容をコピー＆ペーストして保存します。
3. 画面右上の **「デプロイ」** > **「新しいデプロイ」** をクリックします。
4. 種類を **「ウェブアプリ」** に選択します。
5. 設定を以下のように変更します：
   - **次のユーザーとして実行**: 自分
   - **アクセスできるユーザー**: 全員（※ここを「全員」にしないとGitHub Pagesから接続できません）
6. **「デプロイ」** をクリックし、承認を求められたら許可します。
7. 発行された **「ウェブアプリのURL」** をコピーします。

## 3. アプリケーションの設定
1. プロジェクトフォルダにある `config.js` を開き、先ほどコピーしたURLを `API_URL` に貼り付けます。

```javascript
export const config = {
    API_URL: ""
};
```

これで設定は完了です！
