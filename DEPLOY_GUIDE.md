# Netlify デプロイ & スマホアプリ化ガイド

このガイドでは、作成した「日用品ストック管理ツール」をNetlifyで公開し、スマートフォンのホーム画面に追加して「スマホアプリ」として使う手順を説明します。

## 1. 公開用ファイルの準備
パソコン上の以下の6つのファイルを、1つの新しいフォルダ（例：「stock-app」）にコピーしてください：

- `index.html`
- `style.css`
- `app.js`
- `manifest.json`
- `sw.js`
- `(もしあればアイコン画像ファイル)`

※ `server.ps1` はサーバー用なので不要です。

## 2. Netlifyへのアップロード
1. [Netlify (netlify.com)](https://www.netlify.com/) にアクセスし、無料アカウントを作成またはログインします。
2. ログイン後のダッシュボードで、**「Add new site」** ＞ **「Deploy manually」** を選択します。
3. 画面中央の点線枠（"Drag and drop your site folder here"）に、先ほど作成した**「フォルダごと」**ドラッグ＆ドロップします。
4. 数秒でデプロイが完了し、公開用URL（例：`https://random-name-123.netlify.app`）が表示されます。

## 3. スマホへの「アプリ」インストール
1. スマートフォンのブラウザ（AndroidならChrome）で、発行されたNetlifyのURLを開きます。
2. ブラウザのメニュー（右上または右下の「⋮」や「三」ボタン）をタップします。
3. **「ホーム画面に追加」** または **「アプリをインストール」** を選択します。
4. ホーム画面にアイコンが追加され、以降はアイコンをタップするだけで、パソコン不要の**単体スマホアプリ**として動作します。

## 注意事項
- データはスマホの `localStorage` に保存されるため、NetlifyのURLが変わらない限り、データは保持されます。
- 個人利用のため、インターネット上に広く公開したくない場合は、Netlifyのサイト設定で「Password protection」（有料）をかけるか、作成したURLを誰にも教えないようにしてください。
